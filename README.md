## 📂 Repository Architecture & Workspace Rules

To keep the repository clean and maintain professional software development guidelines, each day's task operates inside its own isolated folder container. 

Every standalone module automatically contains:
* 📄 **Task Dashboards (`.md`)** — Outlining the specific daily documentation, target framework analysis, and data mapping.
* 📝 **Prompt Resource Logs (`.txt`)** — Tracking raw input syntax configurations used across baseline and comparison testing.
* 📋 **Output Sheets (`.md` / `.txt`)** — Preserving response structures streaming natively from the AI engine.
* 🖼️ **Visual Assets (`.png` / `.jpg`)** — Screen-captures verifying extension interfaces, token limits, and premium vector infographics.
