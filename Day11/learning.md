# Core Engineering Learnings: Day 11 ATS Mechanics

* **Keyword Density Optimization:** Understood how modern enterprise HR parsers hunt for exact phrases like "Relational Normalization" or "Algorithm Constraints". Integrating these into project scripts naturally scales indexing visibility.
* **Factual Constraint Engineering:** Practiced modifying the contextual weight of an engineering profile (focusing on system logic execution) without adding fabricated internships, fake dates, or unverified skills.
* **SaaS UI Verbs Mapping:** Learned that replacing generic words with active deployment terms (e.g., transforming "worked on a project" to "Architected relational schema instances") significantly enhances resume indexing and readability.
