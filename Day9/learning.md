# Strategic Learnings: Day 9 Iterative AI Development

* **Prompt Engineering for App Scaling:** Asking AI to build multi-tier calculation layers in a single run often leads to code termination or partial blocks. Iterating feature additions onto an existing codebase maintains script stability.
* **Client-Side CSV Parsing:** Learned how to safely process raw tabular arrays via internal text readers and integrate external file objects directly into an active web dashboard application without using a database backend.
* **Component Optimization:** Understood how decoupling component rules for calculators prevents UI charts from breaking when data records expand in size.
